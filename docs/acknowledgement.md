# OSL Status Acknowledgement

Accepted projects should acknowledge OSL status in `README.md` and, where
applicable, public documentation. The acknowledgement helps users and
contributors understand the relationship between OSL and the project.

## Allowed phrases

These phrases are appropriate when they match the current status:

- Incubated by Open Science Labs
- Part of the OSL Incubator Program
- Graduated from the OSL Incubator Program

## Avoid unless separately agreed

Do not use these phrases unless OSL explicitly approves them for a specific
context:

- OSL certified
- OSL guaranteed
- OSL maintained
- Official OSL software
- OSL approved scientific tool

## PoC wording

```markdown
## Open Science Labs Proof of Concept

This project is participating in the Open Science Labs (OSL) Incubator Program
Proof of Concept stage. PoC means the project is exploring feasibility,
usefulness, and fit with OSL values. Day-to-day maintenance, roadmap decisions,
and releases remain the responsibility of the project maintainers.
```

## Incubation wording

```markdown
## Open Science Labs Incubation

This project is incubated by Open Science Labs (OSL). Incubation means that OSL
supports the project through mentorship, structure, community connection, and
sustainability guidance, while day-to-day maintenance, roadmap decisions, and
releases remain the responsibility of the project maintainers.
```

## Graduation wording

```markdown
## Open Science Labs Graduation

This project graduated from the Open Science Labs (OSL) Incubator Program. It
continues to be maintained by its project maintainers. Graduation means the
project completed its OSL incubation milestones and can operate with less direct
OSL incubation support.
```

## Previously incubated or archived wording

```markdown
This project was previously part of the Open Science Labs (OSL) Incubator
Program but is no longer listed as active by OSL. Please see the project status
or maintainer notes for current maintenance information.
```

## When to update

Update acknowledgement text when a project:

- moves from PoC to Incubation;
- graduates;
- moves outside an OSL organization;
- is paused or marked at risk;
- is removed from public OSL lists;
- is archived;
- is reactivated.
